# ศูนย์บริหารจัดการน้ำอัจฉริยะ — เขตอุตสาหกรรม (Demo Dashboard)

Dashboard ส่วนกลางสำหรับบริหารจัดการน้ำร่วมกันของหลายโรงงานในเขตอุตสาหกรรม
เป้าหมาย: ควบคุมคุณภาพน้ำ · ป้องกันน้ำท่วม · ลดต้นทุนการใช้น้ำและพลังงาน

แล้วเปิด http://localhost:8000

## ฟีเจอร์

| ส่วน | รายละเอียด |
|---|---|
| แผนที่ (Leaflet) | ศูนย์กลาง 13.984382, 100.555301 · 4 ชั้นข้อมูลเปิด/ปิดได้: เซนเซอร์ IoT, จุดเสี่ยงน้ำท่วม (AI), สถานีสูบ SCADA, อาคาร/มิเตอร์ |
| เซนเซอร์ IoT | คุณภาพน้ำ (pH, DO, ความขุ่น, WQI) และระดับน้ำเทียบเกณฑ์วิกฤต — คลิกแถวเพื่อดูบนแผนที่ |
| AI Flood Prediction | คาดการณ์ระดับน้ำล่วงหน้า 12 ชม. รายสถานี พร้อมเส้นเกณฑ์วิกฤตและคะแนนความเสี่ยง |
| SCADA / PLC / Pump | สถานะสถานีสูบ 3 แห่ง, ปั๊มรายเครื่อง (อัตราไหล/แรงดัน/กำลัง), ประตูน้ำ, โรงผลิตน้ำประปา |
| AI Anomaly Detection | ฟีดความผิดปกติข้ามระบบ (เซนเซอร์/ปั๊ม/มิเตอร์/เครือข่าย) พร้อมค่าความเชื่อมั่นของโมเดล |
| Meter Billing System | คำนวณบิลจากปริมาณใช้จริง (ค่าน้ำ + ค่าบำบัด + ค่าบริการ + VAT) — กด "ดูบิล" เพื่อดูใบแจ้งรายอาคาร |
| ธีม Dark / Light | ปุ่มสลับธีมที่ header (จำค่าใน localStorage) — สลับทั้ง UI, กราฟ Chart.js และ basemap ของแผนที่ |

## โครงสร้างโปรเจกต์

```
index.html              โครงหน้าหลัก (ประกอบ partial ทุกส่วน)
css/style.css           design tokens 2 ธีม (dark ค่าเริ่มต้น / light ผ่าน html[data-theme])
data/                   ข้อมูลจำลอง แยกไฟล์ตามชนิด
  sensors.json          เซนเซอร์ IoT คุณภาพน้ำ/ระดับน้ำ
  predictions.json      ผลคาดการณ์ระดับน้ำจาก AI
  scada.json            SCADA / PLC / สถานีสูบ / ประตูน้ำ
  anomalies.json        เหตุการณ์ผิดปกติที่ AI ตรวจพบ
  buildings.json        อาคาร + มิเตอร์ + อัตราค่าน้ำ
partials/               HTML component แยกไฟล์ (โหลดด้วย data-include)
js/
  config.js             ค่าคงที่ / สี / ป้ายสถานะ
  utils.js              ฟังก์ชันช่วยเหลือ (format, badge ฯลฯ)
  dataService.js        โหลด JSON + สูตรคำนวณบิล
  main.js               bootstrap: โหลด partial → โหลดข้อมูล → init ทุก component
  js/components/        component ละไฟล์ (header, kpi, map, prediction, scada, sensor, anomaly, billing)
```

ไลบรารีจาก CDN: Tailwind CSS, Leaflet 1.9.4, Chart.js 4 — ข้อมูลทั้งหมดเป็นข้อมูลจำลองเพื่อการสาธิต
